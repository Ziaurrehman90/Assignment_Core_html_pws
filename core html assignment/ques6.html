HTML TAGS

  1. Paragraph tag : Defines a paragraph.
  2. Comment tag : Defines a comment.
  3. Anchor tag : Defines a hyperlink.
  4. Adress tag : Defines contact information for the author/owner of a document.
  5. Break tag : Defines a single line break.
  6. Divison tag : Defines a section in a document.
  7. Form tag : Defines an HTML form for user input.
  8. Head tag : Contains metadata/information for the document.
  9. Html tag : Defines the root of an HTML document.
  10. Image tag : Defines an image.
  11. Link tag : Defines the relationship between a document and an external resource (most used to link to style sheets)
  12. Ordered List : Defines an ordered list.
  13. Small tag : Defines smaller text.
  14. table tag : Defines a table.
  15. Title tag : Defines a title for the document.
